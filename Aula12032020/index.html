Tags e suas funcionalidades

Essa é a tag "p", utilizada para escrever parágrafos, e logo abaixo, será explicado o uso de outras tags e suas funções.

!DOCTYPE html Ele é como uma marcador para que os navegadores processem o conteúdo corretamente
br Quebra a linha
ol Cria uma lista ordenada por números
ul Cria uma lista, "com bolinhas pretas"
i Deixa a palavra caída
hr Cria uma linha na tela
h1,h2,h3,h4,h5 Esses são os títulos, quanto menor o número, maior o título
body Aonde escrevemos e editamos o site
style Essa tag é utilizada para personalizar o site
html É o que envolve todos os códigos do nosso site
id Identificador de tag
class Lista das classes de um elemento

Obs: Não entendi muito bem como funciona o id e o class.
